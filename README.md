# citizenRとは

citizenRは市民活動団体の情報を一覧で表示し、新規登録・更新・削除ができるアプリケーションです。

# 作成の経緯

# 起動方法

CitizenApplication.java内のmainメソッドを起動し、http://localhost:8080/citizen にアクセスします。

# 開発環境

* 言語　java
* データベース　H2 database
* フレームワーク　spring framework
